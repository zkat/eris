# Kallisti

...to the Prettiest!

(this is mostly a toy project but who knows what might happen)

## Development

### Pre-Requisites

* Rust 1.60+
* NodeJS 16+ (and NPM 8+)
* `rustup target add wasm32-unknown-unknown`

### Build & Run

* `cargo make dev` for auto-reloading dev server
* `cargo make build` for final release
